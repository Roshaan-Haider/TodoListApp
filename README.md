# TodoListApp

## Overview

TodoListApp is a Java-based task management application developed as part of a Software Construction and Development (SCD) project. The application helps users organize daily activities by allowing them to create, manage, prioritize, and track tasks efficiently.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Set task priorities
* Create reminders
* Organize and manage task lists
* User-friendly interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Java GUI Components

## Learning Outcomes

This project was developed to strengthen understanding of:

* Object-Oriented Programming
* Event-driven programming
* User interface development
* Data management
* Software design principles
